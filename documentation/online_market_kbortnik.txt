Заголовок страницы: Items:
На странице должен отображаться список товаров. Также на странице располагается:
– форма, в которой можно выбрать способ фильтрации;
– форма, в которой указывается, сколько товаров нужно отображать на данной странице. Примечание: Если товаров в базе больше, чем в форме "количество товаров на странице", то осуществляется постраничный вывод.
1 Для пользователей с привилегией «Customer»:
у каждого товара в форме есть поле для указания его количества и кнопка "Заказать", после нажатия которой формируется заказ.
2 Для пользователей с привилегией «Sale»:
на странице есть ссылки на страницы "Добавление товара" и "Загрузка товаров".
 • Для каждого товара доступна кнопка "Обновить", при нажатии на которую происходит переход на страницу "Обновление товара".
 • Для каждого товара доступна кнопка "Копировать", при нажатии на которую происходит переход на страницу "Копирование товара".
 • В форме каждого товара есть чекбокс и кнопка "Удалить", расположенная внизу страницы, при нажатии на которую удаляются отмеченные товары.

Заголовок страниц: "Добавление товара": данная страница доступна только для пользователя с правами доступа Sale.
На странице расположена форма, на которой доступны все поля, присущие Items.
• На форме имеется кнопка "Добавить", при нажатии на которую происходит добавление товара.
• На странице имеется кнопка "Отмена", при нажатии на которую пользователя перебрасывает на страницу "Items" без добавления товара.

Заголовок страницы: Загрузка товаров: данная страница доступна только для пользователя с правами доступа Sale.
На странице присутствует форма, в которой можно загрузить файл.

Заголовок страницы: Обновление товара: Данная страница доступна только для пользователя с правами доступа Sale. 
На странице расположена форма, имеющие все поля, присущие Items. Вышеуказанные поля, в свою очередь, заполнены данными предмета, который пользователь обновляет.
• В форме имеется кнопка "Обновить", при нажатии на которую происходит обновление товара.
• На странице имеется кнопка "Отмена", при нажатии на которую пользователя перебрасывает на страницу "Items" без обновления товара.

Заголовок страницы: Копирование товара: Данная страница доступна только для пользователь с правами доступа Sale.
На странице расположена форма, на которой присутствуют все поля, присущие Items. Вышеуказанные поля, в свою очередь, заполнены данными предмета, который пользователь копирует.
• На форме имеется кнопка "Добавить", при нажатии на которую происходит добавление товара.
• На странице имеется кнопка "Отмена", при нажатии на которую пользователя перебрасывает на страницу "Items" без добавления товара.

Заголовок страницы: Заказы: 
На странице должен отображаться список заказов.
Имеется форма, на которой можно выбрать способ фильтрации.
Имеется форма, на которой указывается, сколько заказов нужно отображать на данной странице. Примечание: Если заказов в базе больше, чем в форме "Количество заказов на странице", то осуществляется постраничный вывод.
Для пользователя с привилегией «Sale»:
• Должны отображаться заказы всех пользователей.
• Для каждого заказа есть возможность изменить статус и сразу его обновить по нажатию кнопки "Обновить" в форме заказа.
Для пользователей с привилегией «Customer»: 
• Должны отображаться их заказы.

Заголовок страницы: Статьи: 
На странице отображается список статей. Статьи отсортированы по дате добавления по убыванию. Имеется форма, в которой можно указать, сколько статьей должно отображаться на странице. Примечание: Если статей в базе больше, чем указано "Количество статей на странице", то должен осуществляться постраничный вывод. 
При нажатии на заголовок статьи происходит переход на страницу статьи. Под каждой статьей расположены комментарии.
Доступно только пользователям с правами доступа Sale: 
• Имеется кнопка "Добавить статью" с переходом на страницу "Добавление статьи". 
• Возле каждой статьи есть чекбокс, и при нажатии кнопки «Удалить» под списком статей можно удалить все отмеченные статьи.

Заголовок страницы: Добавление статьи: Страница доступна только пользователю с правами доступа Sale.
На странице имеются поля: 
• Заголовок статьи 
• Текст статьи 
• Кнопка "Сохранить" для добавления статьи, а также кнопка "Отмена", при нажатии на которую происходит возврат на страницу статьи.

Заголовок страницы: Изменение статьи: Страница доступна только пользователю с правами доступа Sale.
На странице имеются предзаполненные поля, которые заполнены данными той статьи, которую пользователь изменяет.
Также имеются кнопки "Сохранить" с возвращением к списку статей с сохранением изменений, кнопка "Отмена" – с возвращением к списку статей без внесения изменений.

 Заголовок страницы: Профиль: Данная страница доступна только для пользователя с правами доступа Customer 
На данной странице имеются поля, на которых присутствует вся информация, присущая для страницы профиля. Также имеются кнопки:
• "Изменить данные" с переходом на страницу "Изменение данных".
• "Сменить пароль" с переходом на страницу "Смена пароля".

Заголовок страницы: Изменение данных: Данная страница доступна только для пользователя с правами доступа Customer
На данной странице имеется форма, на которой присутствуют все поля, присущие для изменения профиля. Также на данной странице имеется кнопка "Сохранить" и "Отмена".

Заголовок страницы: Смена пароля: Данная страница доступна только для пользователя с правами доступа Customer
На странице присутствует форма с полями ввода для нового и старого паролей, а также две кнопки – "Изменить пароль" и "Отмена".
